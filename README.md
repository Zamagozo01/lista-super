# Lista de Súper Interactiva

Esta es una página web sencilla para crear una lista de súper tipo catálogo.

## Funciones

- Ver alimentos por categoría
- Buscar ingredientes
- Seleccionar cantidades con botones de más y menos
- Copiar la lista
- Exportar la lista en formato .doc

## Cómo usar

Abre la página publicada con GitHub Pages y selecciona los ingredientes que quieras agregar a tu lista.

El archivo principal del proyecto es:
